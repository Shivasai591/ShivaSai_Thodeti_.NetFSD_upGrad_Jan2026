## Project Overview
This is a multi-page E-Learning platform built using HTML, CSS, Bootstrap, and JavaScript.

## Features
- Dashboard with progress tracking
- Courses page with completion system
- Dynamic Quiz (15 questions)
- Profile page with user name input
- LocalStorage for saving data
- Async JavaScript (Promise, async/await, setTimeout)
- Jest testing

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- Jest

## How to Run
1. Open project in VS Code
2. Use Live Server
3. Run `npm test` for testing
